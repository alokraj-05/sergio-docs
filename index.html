Sergio Discord Bot Documentation

Setup

Follow these steps to set up the Sergio Discord Bot:

  1. Invite the bot to your server using this invite link.
  2. Ensure the bot has the necessary permissions.
  3. Use the !setup command to initialize the bot in your server.

Commands

Here are some of the main commands for Sergio Bot:

  • !help - Display a list of available commands
  • !ping - Check the bot's response time
  • !info - Get information about the bot
  • !settings - Adjust bot settings for your server

Prefix

The default prefix for Sergio Bot is !. You can change it using the following command:

!setprefix [new_prefix]

For example, to change the prefix to $, use:

!setprefix $

Features

Sergio Bot comes with a variety of features to enhance your Discord server:

  • Moderation tools (kick, ban, mute)
  • Custom welcome messages
  • Role management
  • Music playback
  • Trivia games
  • And much more!

To learn more about each feature, use the !help [feature_name] command.
