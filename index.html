Sergio Docs

Welcome to Sergio Bot! While there are numerous moderation bots available for Discord, Sergio offers a unique approach to server management. What sets Sergio apart isn't an abundance of extra features, but rather its focus on providing more detailed and customizable commands. This allows for greater specificity in moderation tasks, tailoring the bot's responses to your server's particular needs. You can use Sergio alongside other bots, or as your primary moderation tool. Moreover, as an open-source project, you have the freedom to modify Sergio, adding or adjusting commands to perfectly suit your community's requirements. Choose Sergio for a moderation experience that prioritizes precision and customization.

Setup

Follow these steps to set up the Sergio Discord Bot:

  1. Invite the bot to your server using this invite link.
  2. Ensure the bot has the necessary permissions.

Commands

Sergio Bot supports two types of commands:

  1. Slash Commands
  2. Prefix Commands

Slash Commands

Slash commands can be used at any time by typing / in the chat, followed by the command name. These commands are always available and do not require a prefix.

Prefix Commands

The default prefix for Sergio Bot is S! (case-sensitive). You can use prefix commands by typing the prefix followed by the command name. For example:

  1. S!help

    // Use this command to get the all the prefix comamnds.

  2. S!sp !

    // you can even change the prefix by using setprefix (sp)

  3. S!ping

    // Gives the ping of user

  4. S!ban

    // Bans a user (PRE-REQUISITE: the user who is executing the command at least have {Aministrator, Ban-members} Perms.)

  5. S!user @mentionuser

    // Gets the user data

  6. S!lock

    // Make the channel private

  7. S!unlock

    // Unlock the channel and make it public.

  8. S!fb

    // Fetch all the banned user.

  9. S!sn

    // Set Nickname for the user (PRE-REQUISITE: {Admin Perms for the user and highest role for the bot.})

  10. S!si

    // Gives you the all possible guild info.

Remember, the default prefix S! is case-sensitive, so make sure to use the correct capitalization when entering commands.

If you ever need assistance with commands, you can use the help command with either method:

  • Slash command: /help
  • Prefix command: S!help (or your custom prefix)

Features

Sergio Bot comes with a variety of features to enhance your Discord server:

Feature Sub-commands
Moderation
  • Ban
  • Kick
  • Mute
  • Timeout
  • Anti-link
  • Anti-spam
Custom Welcome Messages
  • Set welcome message
  • Set welcome channel
  • Enable/disable welcome messages
Role Management
  • Add role
  • Remove role
  • Create role
  • Delete role
  • Role info
Music Playback
  • Play
  • Pause
  • Skip
  • Queue
  • Now playing
Social Notifcations
  • Set Youtube Notifcations
  • Remove Youtube Notifcations
  • Set Github Update Notifcations
  • Set Twitch Update Notifcations

To learn more about each feature and its sub-commands, use the S!help [feature_name] command or the slash command /help [feature_name].
